📌 Project Summary

In this project, I developed a simple 3D study desk environment using Blender. Starting from basic shapes, I built a structured scene that includes a table, laptop, books, and a mug.

I focused on object placement, scaling, and composition to make the scene look organized and realistic. Basic lighting and materials were applied to enhance the overall visual quality and give the scene a complete look.

This project helped me understand how to convert a basic idea into a complete 3D environment using fundamental Blender tools.
